Twitch Custom API for BOTs

Nightbot 등지에서 적당히 써먹기 위한 용도로 JavaScript를 이용한 커스텀 API 제작 연습


1. 업타임 표시 - http://cruxtar.byethost11.com/uptime.html?username={CHANNEL}

기능 및 사용 방법:

현재 진행 중인 방송의 업타임(방송 진행한 시간)을 표시.

{CHANNEL}에 원하는 채널명(트위치 ID)을 기입하여 사용한다.

옵션: (API 주소 뒤에 덧붙여서 사용)

&offline={MESSAGE}

방송 중이지 않을 때 메시지를 {MESSAGE}로 변경.

기본값은 Channel is not live.

&units={NUMBER(1~4)}

표시할 단위의 범위를 지정.

값이 1일 경우 일 단위, 2일 경우 시간 단위, 3일 경우 분 단위, 4일 경우 초 단위까지 출력한다.

기본값은 4(초 단위까지)이며, 값이 4보다 클 경우 4를 입력했을 때와 같은 결과를 출력한다.

&customunit={DD-HH-MM-SS-DDs-HHs-MMs-SSs}

일, 시간, 분, 초의 단위를 DD, HH, MM, SS로 변경.

DDs, HHs, MMs, SSs는 복수형 단위로 입력하지 않아도 되며 미입력 시 DD, HH, MM, SS와 같은 값이 된다.

기본값은 day, hour, minute, second(복수일 경우 복수형 단위 사용)

예제:

http://cruxtar.byethost11.com/uptime.html?username=deadheat7&offline=방송이 종료되었습니다.&units=3&customunit=일-시간-분-초

deadheat7 채널이 3시간 5분 25초째 방송 중일 때 출력 결과: 3시간 5분

← "&units=3": 분 단위까지 표시 / "customunit=일-시간-분-초": 일, 시간, 분, 초 단위로 표시


deadheat7 채널이 오프라인일 때 출력 결과: 방송이 종료되었습니다.

← "&offline=방송이 종료되었습니다."로 입력된 오프라인 알림 메시지 표시)
